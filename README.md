#　阳光英语 (类似于刷抖音的英语学习 app)

## 使用的技术栈

1.  vue 全家桶
2.  vant-ui 组件库

## 项目初始化

1.  使用 vue-cli 创建项目
2.  导入 vant 组件库(按需导入)
    - 在 babel.config.js 文件中设置按需导入
    - 在 assets 文件夹的 vant 文件中创建 index.js 导入需要使用的 vant 组件
    - 在 main.js 中执行导入 vant 组件的主文件
3.  导入 css 初始化文件 base.css
4.  在 github 上创建一个远程仓库 sunshine 并把创建好的项目提交上去
